## Description:
---------------

A mod from Clan REO

Sir Renal is in charge of negotiating the items of the Order of the Most Radiant Heart and can be found within the order HQ (area = ar0903 X = 1224 Y = 477).

Renal, besides offering services typical of temples, sells the following new goods:

Ring of Wisdom
Ring of Greater Holiness
Sacred Helm
Sacred Plate Mail
Chain Mail +1: Protected against Evil
Mace of the Celestial Fire
Bastard Sword +1: Justice
Sir Errant's Virtue
False Dragonscale Armor +3: Blue Mountain
Medium Shield +2: Magebane



## Install:
--------

Extract archive contents into your game directory
Run setup-Renal.exe
Follow the prompt
